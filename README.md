# Used Item Price Checker

A simple used item price checker for payout, bottom price, buyer offers, and sold records.

Live site:

https://wkc50283-star.github.io/used-item-price-checker/

## Features

- Check bottom price before selling a used item
- Calculate net payout after fees and costs
- Check buyer offer
- Compare competitor price
- Save sold records in browser
- Download records as CSV
- Supports English, Traditional Chinese, Simplified Chinese, Japanese, and Korean

## Note

This tool is for personal calculation only. It is not an appraisal, receipt, invoice, or guaranteed selling price.
