# 二手商品售價檢查器 / Used Item Price Checker

一個簡單的二手商品售價檢查工具，幫助二手賣家在出售前試算實際入帳、建議底價、買家出價與相關成本。

A simple used item price checker for sellers to calculate net payout, bottom price, buyer offers, and selling costs before listing or accepting an offer.

## Live Site

https://wkc50283-star.github.io/used-item-price-checker/

## 這個工具可以做什麼？

二手商品出售時，賣家常常只看售價，卻容易忽略平台手續費、金流費、運費、包材、清潔、維修、整理成本與買家殺價後的實際入帳。

這個工具可以協助你快速檢查：

- 想賣價格是否合理
- 買家出價是否可以接受
- 扣除平台手續費後的實際入帳
- 運費、包材、清潔、維修與其他成本
- 建議底價
- 競爭價格是否低於底線
- 售出紀錄
- CSV 紀錄下載備份

## Features

- Calculate net payout after fees and costs
- Check bottom price before selling a used item
- Evaluate buyer offers
- Compare competitor price
- Save sold records in browser localStorage
- Download records as CSV
- Supports English, Traditional Chinese, Simplified Chinese, Japanese, and Korean

## 適合誰使用？

- 二手商品賣家
- 網拍賣家
- 個人轉售者
- Facebook Marketplace 使用者
- eBay / Mercari / 蝦皮 / 露天等平台使用者
- 想避免賣太低或被亂殺價的人

## 使用方式

1. 輸入商品名稱
2. 輸入想賣價格
3. 輸入最低想拿回金額
4. 輸入平台手續費、運費與其他成本
5. 如果有買家殺價，輸入買家出價
6. 查看建議底價、實際入帳與判斷結果
7. 商品售出後，可加入售出紀錄
8. 需要備份時，可下載 CSV

## 注意事項

本工具僅供個人試算使用，不是正式估價工具、行情判斷工具、收據、發票或保證成交價格。

所有價格、成本與是否出售，仍由使用者自行判斷。

This tool is for personal calculation only. It is not an appraisal, receipt, invoice, or guaranteed selling price.
